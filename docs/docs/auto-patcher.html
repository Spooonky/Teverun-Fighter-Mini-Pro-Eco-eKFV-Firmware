Auto Patcher (Offline-Tool)

Die Variante zum Herunterladen mit Menüführung

Am komfortabelsten erfolgt die Bearbeitung über den Web-Patcher direkt im Browser, ohne Installation. Wer das Werkzeug stattdessen offline auf dem PC nutzen möchte, findet hier die erforderlichen Schritte. Beide Wege erzeugen identische Dateien.

1. Vorbereiten

1

Laufzeitumgebung einrichten

Installieren Sie Python und setzen Sie im Installer den Haken bei „Add Python to PATH". Weitere Voraussetzungen bestehen nicht.

2

FIN bereithalten

Für die Namensänderung benötigen Sie Ihre FIN. Den Fundort beschreibt FIN / VIN auslesen.

2. Tool starten

3

Befehl im Verzeichnis des Tools ausführen

python -m teverun_patcher.cli

Anschließend führt Sie eine menügesteuerte Eingabe durch sämtliche Optionen.

3. Menüführung

4

Firmware wählen

Wählen Sie die zu modifizierende Firmware zur Anpassung oder die ALI-Original-Firmware zum reinen Aufspielen.

5

Optionen konfigurieren

Im Hauptmenü stehen folgende Optionen zur Auswahl:

  • Geschwindigkeit freigeben: hebt die werkseitige Begrenzung auf (Feineinstellung anschließend im Display).
  • Bluetooth-Name ändern: FIN eingeben und die gewünschte Länderkennung wählen. Bedeutung der Kennungen: Übersicht.

Die zuletzt getroffene Auswahl wird gespeichert und beim nächsten Start erneut vorgeschlagen.

6

Datei erstellen

Mit b rufen Sie eine Übersicht auf und bestätigen die Auswahl. Anschließend liegen die fertigen Dateien im Verzeichnis builds. Für das Update über die App benötigen Sie die HEX-Datei.

Dateiname nicht ändern: Benennen Sie die Datei nicht um. Die App prüft den Dateinamen exakt. Eine Umbenennung kann den Roller beim Aufspielen beschädigen.
Kompatibilität: Prüfen Sie vor dem Aufspielen die Kompatibilitätsübersicht.
Rechtlicher Hinweis: Ohne Geschwindigkeitsbegrenzung erlischt die Betriebserlaubnis. Zulässig ist die Nutzung ausschließlich auf privatem Gelände.
