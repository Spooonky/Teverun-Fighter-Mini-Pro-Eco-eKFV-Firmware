Browser-Tool

Web-Patcher (Beta)

Firmware direkt im Browser erstellen, ohne Installation

Wählen Sie Ihre Firmware, konfigurieren Sie die Optionen und laden Sie die fertige Datei herunter. Die gesamte Verarbeitung erfolgt lokal im Browser. Eine Installation ist nicht erforderlich, und es werden keine Daten übertragen. Alle Daten verbleiben auf Ihrem Gerät.

Wichtig

Dateiname nicht ändern: Benennen Sie die erstellte Datei nicht um. Die Teverun-App prüft den Dateinamen exakt. Eine Umbenennung kann den Roller beim Aufspielen beschädigen.
Kompatibilität: Prüfen Sie vor dem Aufspielen die Kompatibilitätsübersicht.
Zero-Start / Kick-Start: Diese Funktion lässt sich nicht per Firmware freischalten. Sie ist im Motorcontroller verankert und beim Fighter chargenabhängig (erste Charge: nein, ab der dritten Charge ab Werk). Weiterführende Informationen finden Sie in den FAQ.
