Anleitung

Geschwindigkeit freigeben

Die werkseitige Begrenzung aufheben

Der Teverun Fighter Mini Eco / Pro eKFV verfügt ab Werk über eine fest hinterlegte Geschwindigkeitsbegrenzung. Diese Anleitung beschreibt deren Freigabe. Die erforderliche Datei erstellen Sie am komfortabelsten mit dem Web-Patcher.

Rechtlicher Hinweis: Ohne Geschwindigkeitsbegrenzung erlischt die Betriebserlaubnis nach eKFV. Ein so modifizierter Roller darf nicht im öffentlichen Straßenverkehr betrieben werden. Zulässig ist die Nutzung ausschließlich auf privatem, abgesperrtem Gelände und auf eigene Verantwortung.

1. Datei erstellen

Die Freigabe im Web-Patcher konfigurieren.

1

Web-Patcher öffnen

Öffnen Sie den Web-Patcher, wählen Sie „Patcher laden & starten" und bestätigen Sie den Haftungsausschluss.

2

Firmware wählen

Wählen Sie die passende Firmware. Unterstützt werden D5 und R5 (neueste Version: R5.4.19). Die Auswahl steht Ihnen frei. Wenn Sie noch eine ältere Firmwareversion einsetzen, können Sie damit auf eine neuere Version aktualisieren.

3

Freigabe auswählen

Stellen Sie Geschwindigkeit freigeben auf „Freigeben".

4

Datei erstellen

Wählen Sie Datei erstellen und laden Sie die HEX-Datei herunter. Belassen Sie den Dateinamen unverändert.

2. Aufspielen

Die Datei über die Teverun-App auf den Roller übertragen.

5

In der App flashen

Verbinden Sie den Roller in der Teverun-App, öffnen Sie „System Upgrades", wählen Sie die erzeugte HEX-Datei und starten Sie den Vorgang. Die ausführliche Anleitung Firmware aufspielen gilt unverändert.

Dateiname nicht ändern: Die App prüft den Dateinamen exakt. Eine Umbenennung kann den Roller beim Aufspielen beschädigen.

3. Höchstgeschwindigkeit festlegen

Der entscheidende abschließende Schritt.

6

Im Display-Menü konfigurieren

Die Freigabe hebt lediglich die Begrenzung auf. Die tatsächliche Höchstgeschwindigkeit pro Gang legen Sie anschließend im versteckten Menü Ihres Displays fest. Bleibt der Roller danach scheinbar gleich schnell, liegt die Ursache nahezu immer in diesem Schritt.
