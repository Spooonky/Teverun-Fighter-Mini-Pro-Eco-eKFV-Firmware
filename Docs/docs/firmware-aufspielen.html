Anleitung

Firmware aufspielen

Eine fertige Datei über die Teverun-App auf den Roller übertragen

So übertragen Sie eine fertige HEX-Datei auf Ihren Roller, unabhängig davon, ob Sie die Geschwindigkeit freigegeben oder eine andere Änderung vorgenommen haben. Für die Änderung des Bluetooth-Namens sind anschließend zusätzliche Schritte erforderlich; diese sind in der eigenen Anleitung BLE-Namen ändern beschrieben.

Gefahr eines Bricks: Der Vorgang darf nicht unterbrochen werden. Lassen Sie das Display eingeschaltet, die App geöffnet und schalten Sie den Roller nicht aus. Ein Abbruch kann die Steuereinheit dauerhaft zerstören.

1. Vorbereiten

Prüfen Sie vorab die Voraussetzungen.

1

App installieren

Laden Sie die offizielle Teverun-App aus dem App-Store, installieren Sie sie und melden Sie sich an.

2

Roller verbinden

Schalten Sie den Roller ein, wählen Sie in der App das Bluetooth-Symbol, selektieren Sie Ihr Fahrzeug und warten Sie eine stabile Verbindung ab.

3

Kompatibilität bestätigen

Öffnen Sie in der App „Product Information". Die IVCU-Hardware muss V5.X (Box C) sein. Bei Abweichung brechen Sie sofort ab. Siehe Kompatibilitätsübersicht.

2. Aufspielen

Unterbrechen Sie die Verbindung während des Vorgangs nicht.

4

Update starten

Wählen Sie in der App „System Upgrades", selektieren Sie die erstellte HEX-Datei und wählen Sie „Start". Der Vorgang kann bis zu 13 Minuten dauern.

Dateiname nicht ändern: Benennen Sie die Datei nicht um. Die App prüft den Dateinamen exakt. Eine Umbenennung kann den Roller beim Aufspielen beschädigen.
5

Abschluss abwarten

Warten Sie, bis die App „Upgrade Done" anzeigt. Erst dann ist die Firmware vollständig geschrieben.

Haben Sie den Bluetooth-Namen geändert? Dann folgen die zusätzlichen Schritte unter BLE-Namen ändern.
